#  University Website project
A responsive web project built using HTML and CSS to present structured information about a university, including campus sections, facilities, and student resources.

---

## Live Demo

https://Ayush570566.github.io/Chitkara-University-Website/

---

## Overview

This project focuses on creating a clean and navigable interface for users to explore different aspects of a university. It emphasizes layout clarity, simple navigation, and visual content presentation.

---

## Features

* Responsive layout
* Multi-page navigation
* Structured content sections
* Image-based interface
* Clean and minimal UI

---

## Tech Stack

* HTML
* CSS
